## What is Uxie?

Uxie is a comprehensive PDF reader application that combines powerful annotation tools with cutting-edge AI features to transform how you learn from documents. Whether you're a student, researcher, or professional, Uxie provides everything you need to read, annotate, and understand PDFs more effectively.

## Key Features

<AccordionGroup>
  <Accordion title="📝 Advanced Note-Taking" icon="pen-to-square">
    Built on BlockNote editor with rich formatting, custom blocks, and markdown export. Your notes sync in real-time and integrate seamlessly with highlights.
  </Accordion>
  
  <Accordion title="🤖 AI-Powered Learning" icon="robot">
    Leverage Langchain and Pinecone for intelligent document chat, automatic summarization, and context-aware responses using RAG (Retrieval-Augmented Generation).
  </Accordion>
  
  <Accordion title="🎯 Smart Flashcards" icon="cards-blank">
    Generate flashcards automatically from your documents, answer questions, and receive AI-powered feedback to improve your understanding.
  </Accordion>
  
  <Accordion title="🔊 Text-to-Speech" icon="volume">
    Listen to your documents with sentence-by-sentence highlighting. Supports both local AI models and browser-native TTS.
  </Accordion>
  
  <Accordion title="👁️ OCR Support" icon="scan">
    Extract text from scanned documents and images using advanced OCR technology (English language support).
  </Accordion>
  
  <Accordion title="📖 Multiple Reading Modes" icon="book-open">
    Choose from bionic reading, RSVP speed reading, read-along mode, or full-screen immersive view.
  </Accordion>
</AccordionGroup>

## Technology Stack

Uxie is built with modern technologies to ensure performance, reliability, and scalability:

- **Next.js 14** - React framework with App Router and Server Components
- **tRPC** - End-to-end typesafe APIs
- **Prisma & Supabase** - Database ORM and PostgreSQL hosting
- **Langchain & Pinecone** - AI chat with vector embeddings
- **Liveblocks** - Real-time collaboration infrastructure
- **BlockNote** - Powerful block-based note editor
- **Shadcn UI** - Beautiful, accessible component library

<Note>
  Uxie started as a hackathon project and has evolved into a feature-rich learning platform. It continues to be actively developed with new features and improvements.
</Note>
